# Shieldy

> **_Another TOTP App_**
>
> _Shieldy is a TOTP App. Created using Nuxt4_

----

## Setup

Make sure to install dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
